Technologies & Stack

ESP32

Dual-core microcontroller with Wi-Fi/BLE for IoMT edge nodes, real-time patient monitoring and medical asset tracking.

Arduino

Prototyping biomedical sensors and rapid embedded systems for wearable health devices and data acquisition.

Node.js

Backend microservices for handling IoMT data streams, REST APIs and real-time hospital device coordination.

Firebase

Real-time database & authentication for asset tracking dashboards and secure health data synchronization.

React

Dynamic front-end dashboards for hospital analytics, predictive insights and equipment health visualization.

MQTT

Lightweight IoMT protocol for low-latency communication between medical devices and central analytics.

Python

AI/ML models for predictive maintenance, anomaly detection in biomedical signals and patient risk scoring.

AI/ML

Deep learning & computer vision for diagnostic assistance and real-time medical imaging analytics.
